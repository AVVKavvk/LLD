# Adapter

---

- Definition: Allows objects with incompatible interfaces to collaborate.
- Use Case: You have a "Round Hole" system, but you need to fit a "Square Peg" (legacy code or third-party library) into it.

- What it solves: Allows classes with incompatible interfaces to work together.

- What it prevents: Having to rewrite existing, thoroughly tested code just to make it fit a new system.
