# Decorator

---
- Definition: Lets you attach new behaviors to objects by placing these objects inside special wrapper objects that contain the behaviors.

- Use Case: Formatting text (Bold, Italic), adding toppings to a coffee order.

- What it solves: Lets you attach new behaviors to objects dynamically by placing them inside special wrapper objects.

- What it prevents: "Subclass explosion" (e.g., having to create classes like CoffeeWithMilk, CoffeeWithSugar, CoffeeWithMilkAndSugar).
